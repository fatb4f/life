# Module 2: Operations & Order of Operations

## Big Ideas

- Order of operations (PEMDAS)
- Properties (commutative, associative, distributive)
- Negatives and parentheses

## Key Formulas

- $a(b+c)=ab+ac$
- $(-a)b=-(ab)$
- Evaluate exponents before multiplication/division

## Worked Examples

- **Order of operations.** Evaluate $3-2^3\cdot(4-1):\ 2^3=8,\ 4-1=3,\ 8\cdot 3=24,\ 3-24=-21$.
- **Distribute.** Simplify $2(x-3)+4x=2x-6+4x=6x-6$.

# Module 6: Ratios, Proportions & Graphs

## Big Ideas

- Ratio & rate
- Proportional relationships
- Slope-intercept form
- Reading simple graphs/tables

## Key Formulas

- $\frac{a}{b}=\frac{c}{d}\Rightarrow ad=bc$
- $y=mx+b$; slope $m=\frac{\Delta y}{\Delta x}$

## Worked Examples

- **Find slope.** Through $(2,5)$ and $(6,9)$: $m=\frac{9-5}{6-2}=\frac{4}{4}=1$
- **Set up proportion.** $\frac{3}{5}=\frac{x}{20}\Rightarrow 5x=60\Rightarrow x=12$

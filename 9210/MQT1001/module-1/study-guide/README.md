# Module 1: Real Numbers & Number Line

## Big Ideas

- Sets of numbers (ℕ, ℤ, ℚ, ℝ)
- Absolute value & distance
- Interval notation
- Rounding & significant figures

## Key Formulas

- $\mathbb{N}\subset\mathbb{Z}\subset\mathbb{Q}\subset\mathbb{R}$
- $|a-b|$ is the distance between $a$ and $b$ on the number line.
- Interval notation examples: $[a,b],\ (a,b],\ [a,b),\ (a,b)$

## Worked Examples

- **Distance on the number line.** Find the distance between $-3$ and $5$: $|(-3)-5|=|-8|=8$.
- **Percent to decimal.** Convert $7.5\%$ to decimal: $7.5\%=\frac{7.5}{100}=0.075$.

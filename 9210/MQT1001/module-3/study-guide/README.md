# Module 3: Fractions, Decimals, Percents

## Big Ideas

- Converting between forms
- Simplifying fractions
- Percent change & applications

## Key Formulas

- $\frac{a}{b}=a\div b$
- $\text{percent}=\text{decimal}\times 100\%$
- $\text{percent change}=\frac{\text{new}-\text{old}}{\text{old}}\times 100\%$

## Worked Examples

- **Fraction to decimal.** $\frac{7}{8}=0.875$
- **Percent change.** From 50 to 65: $\frac{65-50}{50}\times 100\%=30\%$.

# Module 5: Linear Equations & Inequalities

## Big Ideas

- Solving one- and two-step equations
- Distributive property & combining like terms
- Inequalities and interval solutions

## Key Formulas

- $ax+b=c\ \Rightarrow\ x=\frac{c-b}{a}$
- Flip inequality when multiplying/dividing by a negative

## Worked Examples

- **Solve linear equation.** $3x-7=11\Rightarrow 3x=18\Rightarrow x=6$
- **Inequality flip.** $-2x>4\Rightarrow x<-2$ (flip when dividing by $-2$)

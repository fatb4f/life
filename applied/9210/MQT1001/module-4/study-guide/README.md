# Module 4: Exponents & Scientific Notation

## Big Ideas

- Laws of exponents
- Negative & fractional exponents
- Scientific notation and sig figs

## Key Formulas

- $a^m a^n=a^{m+n}$,\quad $\frac{a^m}{a^n}=a^{m-n}$,\quad $(a^m)^n=a^{mn}$
- $a^{-n}=\frac{1}{a^n}$,\quad $a^{1/n}=\sqrt[n]{a}$
- $N=a\times 10^k$ with $1\le |a|<10$

## Worked Examples

- **Exponent laws.** $x^3\cdot x^5=x^{8}$
- **Scientific notation.** $0.00056=5.6\times 10^{-4}$

# Phase-1 Workouts (Logging Guide)

- **Quick log:** GitHub Mobile → New Issue → *Workout Log* template.
- **Permanent record:** Copy issue content into `/docs/phase-1/workouts/logs/YYYY-MM-DD_<slug>.md`.
- **Templates:** See `/docs/phase-1/workouts/templates/`.
- **Naming:** `YYYY-MM-DD_<short-title>.md`
- **Goal:** Link training, HRV, and PT notes in one place under version control.
